Oque é linguagem de programação?

Uma linguagem de programação é um conjunto de regras e instruções que permite aos programadores comunicarem-se com computadores, criando programas que realizam tarefas específicas. É como um idioma artificial, onde o código escrito é traduzido para que o computador entenda e execute as ações desejadas.
